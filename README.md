# 演算法學習報告 - 基礎框架

這是一個基於圖片需求所建立的演算法分析報告網頁框架，使用 **React + Vite + Tailwind CSS** 開發。

## 🚀 快速開始

1.  **安裝依賴:**
    ```bash
    npm install
    ```

2.  **啟動開發伺服器:**
    ```bash
    npm run dev
    ```

3.  **修改內容:**
    請編輯 `src/components/` 目錄下的組件來填入你的學習心得：
    *   `AlgorithmIntro.tsx`: 介紹演算法。
    *   `AILearningRecord.tsx`: 紀錄 AI 互動。
    *   `VerificationCorrection.tsx`: 查證與修正。
    *   `VisualExplanation.tsx`: 加入圖表或圖片。
    *   `ApplicationCase.tsx`: 說明實際應用。
    *   `LearningReflection.tsx`: 反思心得。

## 📦 部署至 GitHub Pages

本專案已預先配置好 `gh-pages` 部署工具。

1.  **初始化 Git (如果你還沒做):**
    ```bash
    git init
    git add .
    git commit -m "Initialize project"
    ```

2.  **連結至 GitHub Repository:**
    ```bash
    git remote add origin https://github.com/111316031/RLHF-FINAL.git
    ```

3.  **執行部署命令:**
    ```bash
    npm run deploy
    ```
    這會自動建立專案、將 `dist` 資料夾推送到 `gh-pages` 分支。

4.  **在 GitHub 設定:**
    至 Repository 的 **Settings > Pages**，確保 Source 設定為 `gh-pages` 分支。
